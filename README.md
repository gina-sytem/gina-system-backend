# GraalVM Build
If you want to build with GraalVM make sure to:
- [Have GraalVM installed](https://www.graalvm.org/dev/getting-started/linux) (At least version 21)
- We use Static Native Image, for this [follow this guide to install necessary tooling](https://www.graalvm.org/22.0/reference-manual/native-image/StaticImages/index.html)  
Tip: For build with docker, you just need docker installed
